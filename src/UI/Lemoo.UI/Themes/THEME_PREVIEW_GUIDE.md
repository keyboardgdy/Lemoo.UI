# Lemoo UI 主题预览指南

## 概述

Lemoo UI 提供了多种精心设计的主题方案，每种主题都有其独特的视觉风格和适用场景。本文档将帮助您了解每个主题的特色，以便选择最适合您的主题。

---

## 经典主题

### 1. 原色模式 (Base)
**类别**: Lemoo
**标签**: 默认

**设计理念**:
品牌原色配色方案，展示 Lemoo 品牌特色。

**色彩特征**:
- 主色调: 深色背景 (#1A1E24) 搭配青色强调色 (#00ADB5)
- 风格: 专业、稳重
- 适用场景: 日常使用，展示品牌形象

**预览色彩**:
```
背景:  #1A1E24
表面:  #252B33
强调:  #00ADB5
文本:  #EEEEEE
```

## 创意主题

### 4. 霓虹赛博朋克
**类别**: 创意主题
**标签**: 创新

**设计理念**:
灵感来源于未来的赛博朋克世界，采用高对比度的霓虹色彩与深色背景的碰撞，营造出强烈的科技感和未来感。

**色彩特征**:
- 主色调: 霓虹粉 (#FF006E)、电光蓝 (#00F5FF)、酸性绿 (#39FF14)
- 背景: 深空黑 (#0A0A0F)
- 风格: 前卫、大胆、科技感
- 适用场景: 游戏界面、创意工具、夜间使用

**视觉亮点**:
- 高对比度霓虹色彩
- 霓虹粉色标题栏边框
- 酸性绿科技感边框
- 电光蓝文本色

**预览色彩**:
```
背景:  #0A0A0F
表面:  #0D0D14
强调:  #FF006E (霓虹粉)
文本:  #00F5FF (电光蓝)
边框:  #39FF14 (酸性绿)
```

---

### 5. 极光 (Aurora)
**类别**: 创意主题
**标签**: 优雅

**设计理念**:
灵感来源于北极光的自然奇观，采用流动的渐变色彩和柔和的光晕效果，带来宁静而神秘的视觉体验。

**色彩特征**:
- 主色调: 极光绿 (#00D9A0)、冰晶蓝 (#4FC3F7)、北极紫 (#7C4DFF)
- 背景: 深海黑 (#0F1419)
- 风格: 清冷、优雅、自然
- 适用场景: 专业应用、数据可视化、需要专注的环境

**视觉亮点**:
- 流动的渐变色彩
- 极光绿选中状态
- 冰晶蓝边框点缀
- 霜冻白清新文本

**预览色彩**:
```
背景:  #0F1419
表面:  #141C27
强调:  #00D9A0 (极光绿)
文本:  #E0F7FA (霜冻白)
边框:  #4FC3F7 (冰晶蓝)
```

**色彩心理学**:
- 绿色: 自然、平衡、更新
- 蓝色: 信任、专业、冷静
- 紫色: 创造力、智慧、神秘

---

### 6. 热带夕阳 (Sunset Tropics)
**类别**: 创意主题
**标签**: 温暖

**设计理念**:
灵感来源于热带海滩的夕阳景色，采用温暖的大地色系与活力的珊瑚橙、青绿色的融合，充满热情与生机。

**色彩特征**:
- 主色调: 珊瑚橙 (#FF6B6B)、热带青 (#4ECDC4)、沙滩金 (#FFE66D)
- 背景: 奶油白 (#FFF8F0)
- 风格: 温暖、活力、友好
- 适用场景: 创意应用、社交媒体工具、日间使用

**视觉亮点**:
- 温暖的桃色背景
- 珊瑚橙与热带青的碰撞
- 奶油色柔和基调
- 友好亲和的视觉体验

**预览色彩**:
```
背景:  #FFF8F0 (奶油白)
表面:  #FFE5D9 (桃色)
强调:  #FF6B6B (珊瑚橙)
文本:  #2C3E50
选中:  #4ECDC4 (热带青)
```

**色彩搭配原理**:
- 互补色: 珊瑚橙 (暖) 与热带青 (冷)
- 类比色: 桃色、奶油色、沙滩金
- 明度渐变: 背景色从浅到深的层次

## 主题选择建议

### 按使用场景选择

| 场景 | 推荐主题 | 理由 |
|------|---------|------|
| 日常开发 | 深色模式 | 减少眼睛疲劳，经典配色 |
| 日间办公 | 浅色模式 / 热带夕阳 | 明亮通透，温暖舒适 |
| 游戏应用 | 霓虹赛博朋克 | 强烈的科技感和视觉冲击 |
| 数据分析 | 极光 | 清冷优雅，有助于专注 |
| 创意设计 | 霓虹赛博朋克 / 热带夕阳 | 大胆配色，激发灵感 |
| 专业应用 | 原色模式 / 极光 | 展示品牌形象，专业稳重 |

### 按时间选择

| 时间 | 推荐主题 | 理由 |
|------|---------|------|
| 夜间 | 深色模式 / 霓虹赛博朋克 / 极光 | 深色背景，减少眩光 |
| 日间 | 浅色模式 / 热带夕阳 | 明亮配色，与环境协调 |
| 持续使用 | 极光 / 深色模式 | 柔和配色，眼睛不易疲劳 |

### 按个人喜好选择

| 喜好风格 | 推荐主题 |
|---------|---------|
| 喜欢稳重大方 | 原色模式、深色模式 |
| 喜欢明亮清新 | 浅色模式、热带夕阳 |
| 喜欢前卫科技 | 霓虹赛博朋克 |
| 喜欢优雅自然 | 极光 |
| 喜欢温暖活力 | 热带夕阳 |

## 主题切换

所有主题都可通过 `ThemeManager` 进行切换：

```csharp
// 切换到霓虹赛博朋克主题
ThemeManager.CurrentTheme = ThemeManager.Theme.NeonCyberpunk;

// 切换到极光主题
ThemeManager.CurrentTheme = ThemeManager.Theme.Aurora;

// 切换到热带夕阳主题
ThemeManager.CurrentTheme = ThemeManager.Theme.SunsetTropics;
```

---

## 无障碍设计

所有主题均遵循 WCAG 2.1 无障碍设计标准：

- **对比度**: 主要文本与背景对比度 ≥ 4.5:1 (AA 级)
- **可读性**: 确保在各种光照条件下都能清晰阅读
- **色彩独立性**: 不仅依靠颜色传达信息

## 自定义主题

如果您想创建自己的主题，请参考现有主题的结构：

1. 在 `Themes` 目录下创建新文件夹
2. 创建 `SemanticTokens.xaml` 文件定义颜色
3. 创建主题入口 `.xaml` 文件
4. 在 `ThemeManager.cs` 中注册新主题
5. 在 `.csproj` 文件中添加资源引用

详细步骤请参考各个主题目录下的 `README.md` 文件。
