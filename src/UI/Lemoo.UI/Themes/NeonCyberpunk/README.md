# 霓虹赛博朋克主题 (Neon Cyberpunk Theme)

## 设计理念

霓虹赛博朋克主题灵感来源于未来的赛博朋克世界，采用高对比度的霓虹色彩与深色背景的碰撞，营造出强烈的科技感和未来感。这个主题适合那些喜欢独特、前卫视觉体验的用户。

## 色彩系统

### 主色调
- **霓虹粉** `#FF006E` - 主要强调色，用于选中状态和重要操作
- **电光蓝** `#00F5FF` - 主要文本色和视觉焦点
- **酸性绿** `#39FF14` - 边框和分隔线，营造科技感

### 辅助色
- **紫罗兰** `#8B5CF6` - 次要强调色和悬停效果
- **深空黑** `#0A0A0F` - 主背景色
- **午夜蓝** `#0D0D14` - 次级背景色

### 背景色阶
```
主背景:     #0A0A0F (最深)
次背景:     #0D0D14
卡片背景:   #1A1A25
悬停背景:   #1A1A25
```

## 视觉特征

1. **高对比度** - 霓虹色彩在深色背景上形成强烈对比
2. **发光效果** - 模拟霓虹灯的发光效果（通过颜色亮度实现）
3. **科技感边框** - 使用酸性绿作为边框色
4. **大胆配色** - 不拘泥于传统UI设计规范

## 适用场景

- 游戏界面
- 创意工具
- 夜间使用
- 追求个性化体验的用户

## 设计亮点

- 标题栏边框使用霓虹粉，形成强烈的视觉边界
- 侧边栏选中项使用霓虹粉背景，白色文字确保可读性
- 滚动条悬停时变为霓虹粉，提供交互反馈
- 输入框聚焦时边框变为霓虹粉，清晰指示焦点状态

## 对比度说明

所有主要文本颜色与背景色的对比度均超过 WCAG AAA 标准（7:1），确保在深色背景上的可读性。
