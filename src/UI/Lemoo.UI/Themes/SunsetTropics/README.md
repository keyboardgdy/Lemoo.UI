# 热带夕阳主题 (Sunset Tropics Theme)

## 设计理念

热带夕阳主题灵感来源于热带海滩的夕阳景色，采用温暖的大地色系与活力的珊瑚橙、青绿色的融合，充满热情与生机。这个主题带来温暖、愉悦的视觉体验。

## 色彩系统

### 主色调
- **珊瑚橙** `#FF6B6B` - 主要强调色，象征夕阳的温暖
- **热带青** `#4ECDC4` - 选中状态和重要操作，代表热带海洋
- **沙滩金** `#FFE66D` - 点缀色，如同阳光洒在沙滩上

### 辅助色
- **夕阳红** `#EE5A5A` - 按下状态和强调
- **海蓝** `#2980B9` - 深邃的蓝色元素
- **奶油白** `#FFF8F0` - 主背景色，温暖柔和

### 背景色阶
```
主背景:     #FFF8F0 (奶油色)
次背景:     #FFE5D9 (桃色)
卡片背景:   #FFFBF5 (近白色)
悬停背景:   #FFF0E6 (浅桃色)
```

## 视觉特征

1. **温暖舒适** - 大地色系带来温暖感
2. **活力四射** - 珊瑚橙和热带青的碰撞充满活力
3. **自然和谐** - 模拟自然界的色彩搭配
4. **友好亲和** - 整体色调让人感到轻松愉悦

## 适用场景

- 创意应用
- 社交媒体工具
- 日间使用
- 需要激发灵感的界面

## 设计亮点

- 标题栏使用桃色背景，营造温暖的氛围
- 侧边栏选中项使用热带青背景，与暖色形成活泼对比
- 按钮悬停时使用浅桃色背景，按下时变为珊瑚橙
- 边框和分隔线使用桃色系，保持整体和谐

## 色彩搭配原理

- **互补色运用** - 珊瑚橙（暖色）与热带青（冷色）形成互补
- **类比色和谐** - 桃色、奶油色、沙滩金形成类比色序列
- **明度渐变** - 背景色从浅到深的层次感

## 情感影响

- **温暖感** - 来自桃色和珊瑚橙
- **活力感** - 来自热带青的点缀
- **轻松感** - 来自柔和的背景色
- **愉悦感** - 整体配色带来的积极情绪

## 对比度说明

所有主要文本颜色与背景色的对比度均符合 WCAG AA 标准（4.5:1），确保在浅色背景上的可读性。
