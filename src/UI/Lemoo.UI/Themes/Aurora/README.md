# 极光主题 (Aurora Borealis Theme)

## 设计理念

极光主题灵感来源于北极光的自然奇观，采用流动的渐变色彩和柔和的光晕效果，带来宁静而神秘的视觉体验。这个主题融合了冷色调的优雅与自然的灵动。

## 色彩系统

### 主色调
- **极光绿** `#00D9A0` - 主要强调色，灵感来自北极光的绿色光带
- **冰晶蓝** `#4FC3F7` - 边框和视觉元素，模拟冰晶的清澈感
- **北极紫** `#7C4DFF` - 标签页和次要元素，象征夜空的神秘

### 辅助色
- **霜冻白** `#E0F7FA` - 主要文本色，清新明亮
- **午夜蓝** `#1A237E` - 深邃的背景色
- **深海黑** `#0F1419` - 主背景色

### 背景色阶
```
主背景:     #0F1419 (最深)
次背景:     #141C27
卡片背景:   #1E2936
悬停背景:   #1E2936
```

## 视觉特征

1. **流动感** - 色彩过渡自然柔和
2. **清冷优雅** - 冷色调带来专业感
3. **自然灵感** - 模拟自然光效的色彩搭配
4. **平衡和谐** - 各色彩之间形成和谐的视觉平衡

## 适用场景

- 专业应用
- 数据可视化工具
- 长时间使用的界面
- 需要专注和冷静的工作环境

## 设计亮点

- 侧边栏选中项使用极光绿背景，深色文字形成强烈对比
- 标题栏和边框使用极光绿点缀，如同极光边缘的光晕
- 按钮悬停时使用渐变背景，模拟光晕扩散效果
- 输入框聚焦时边框变为极光绿，清晰且不刺眼

## 色彩心理学

- **绿色** - 象征自然、平衡、更新
- **蓝色** - 代表信任、专业、冷静
- **紫色** - 暗示创造力、智慧、神秘

## 对比度说明

所有文本颜色与背景色的对比度均符合 WCAG AA 标准（4.5:1），主要交互元素达到 AAA 标准（7:1）。
