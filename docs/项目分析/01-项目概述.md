# Lemoo.UI 项目概述

## 项目信息

| 项目名称 | Lemoo - .NET 10 模块化架构框架 |
|---------|------------------------------|
| 项目类型 | 企业级桌面应用框架 + Web API 服务 |
| 开发语言 | C# 13 (.NET 10) |
| 架构模式 | DDD + CQRS + 插件式模块化架构 |
| 许可证 | MIT |

## 项目简介

Lemoo.UI 是一个基于 .NET 10 构建的现代化模块化应用框架。项目采用领域驱动设计（DDD）和命令查询职责分离（CQRS）模式，支持插件式模块架构，可同时构建 WPF 桌面应用和 Web API 服务。

## 核心特性

- **模块化架构**: 插件式模块系统，支持动态加载和依赖管理
- **DDD 支持**: 完整的领域模型支持（聚合根、值对象、领域事件）
- **CQRS 模式**: 命令查询分离，自动验证、缓存、事务管理
- **双宿主支持**: WPF 桌面应用 + Web API 服务
- **现代 UI**: Windows 11 Fluent Design 风格控件库
- **结果模式**: 类型安全的错误处理和函数式编程支持

## 技术栈

- .NET 10 (LTS)

## 快速开始

### 克隆项目
```bash
git clone https://github.com/your-org/Lemoo.UI.git
cd Lemoo.UI
```

### 构建项目
```bash
dotnet build Lemoo.sln
```

### 运行 WPF 应用
```bash
dotnet run --project src/Hosts/Lemoo.Host
```

### 运行 Web API
```bash
dotnet run --project src/Hosts/Lemoo.Api
```
