## 技术栈

### 核心框架
- .NET 10 (LTS)
- C# 13
- Microsoft.Extensions.Hosting

### UI 框架
- WPF (.NET 10-windows)
- CommunityToolkit.Mvvm 8.4.0
- Win11 Fluent Design

### 数据层
- Entity Framework Core 10.0.1
- SQL Server

### 架构实现
- MediatR 14.0.0
- FluentValidation 12.1.1

### 基础设施
- Serilog 4.3.0
- Hangfire
- RabbitMQ.Client
- Redis

## 核心特性

- **模块化架构**: 插件式模块系统，支持动态加载和依赖管理
- **DDD 支持**: 完整的领域模型支持（聚合根、值对象、领域事件）
- **CQRS 模式**: 命令查询分离，自动验证、缓存、事务管理
- **双宿主支持**: WPF 桌面应用 + Web API 服务
- **现代 UI**: Windows 11 Fluent Design 风格控件库
- **结果模式**: 类型安全的错误处理和函数式编程支持
